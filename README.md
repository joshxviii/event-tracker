[**GoLocal**](https://golocalweb.vercel.app) is a web app that let's you view nearby community events!

[![Live Demo](https://img.shields.io/badge/live-demo-brightgreen)](https://golocalweb.vercel.app)  

## Our Goal

Many residents, organizations, and businesses miss out on local events due to scattered information and lack of awareness.  
**Event Tracker** solves this by centralizing event discovery, providing reminders via Google Calendar integration, and fostering community connection through a clean, intuitive platform.

## Key Features

- **User authentication** – Register and log in securely
- **Create & manage events** – Add recurring or one-time events with date, time, location, type, and optional images
- **Powerful filtering** – Search by keyword, event type, or date range
- **Interactive map** – See all nearby events powered by Google Maps API
- **Personal calendar** – View your saved and upcoming events in one place
- **Save to Google Calendar** – Get automatic reminders on your phone
- **Reviews & likes** – Share feedback on past events
- **Friend system** – Connect with other community members (event invites in progress)
- **Dark mode** – Easy on the eyes at night
- **Admin moderation** – Ensures all events follow community guidelines

## Tech Stack

| Layer              | Technology                  |
|--------------------|-----------------------------|
| Frontend           | React.js + JavaScript       |
| Backend / Database | MongoDB (via Azure Cosmos DB) |
| Cloud & Storage    | Azure + Azure Blob Storage  |
| Maps & Calendar    | Google Maps API, Google Calendar API |
| Deployment         | Vercel                      |
